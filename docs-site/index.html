GabyGas en GitHub Pages ✅

Esta versión se publica como sitio estático para que siempre se vea correctamente desde GitHub Pages.

Si hiciste cambios de diseño o agregaste funcionalidades y no los ves aquí, es porque estás abriendo la URL de GitHub Pages (solo muestra este HTML estático).

Nota: la aplicación principal (Next.js + Auth + Prisma + API) requiere un servidor Node.js y base de datos; por eso no puede ejecutarse completa en GitHub Pages.

Ver repositorio Para validar tus cambios reales, ejecuta la app con npm run dev o despliega en Vercel/Render/Railway.
